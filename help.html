Back Of The Napkin

Where Fermi meets your napkin.

Basic Concepts

Back Of The Napkin helps you make rough calculations with uncertain values. Instead of exact numbers, you can use ranges and get statistical results. It's perfect for market sizing, forecasting, and numerical estimations. Inspired by Unsure Calculator .

Input Formats

  • Simple Numbers: Just type the number (e.g., 42)
  • Ranges: Use ~ between numbers (e.g., 40~50)
  • Percentages: Add % symbol (e.g., 20% or 15%~25%)
  • Large Numbers: Use K, M, B, T suffixes
    • K = thousands (e.g., 50K = 50,000)
    • M = millions (e.g., 1.5M = 1,500,000)
    • B = billions (e.g., 2B = 2,000,000,000)
    • T = trillions (e.g., 1T = 1,000,000,000,000)

Examples

Example 1: Business Revenue Estimation

Estimating annual revenue for a business:

  • customers: 1000~1200 (customers per month)
  • spend: 20~30 (dollars per customer)
  • Formula: customers * spend * 12
  • This will calculate the range of annual revenue

Example 2: Project Cost with Uncertainty

Estimating project cost with contingency:

  • base_cost: 100K~120K
  • contingency: 15%~25%
  • Formula: base_cost * (1 + contingency)

Example 3: Market Size Calculation

Calculating total addressable market:

  • population: 5M~6M
  • adoption_rate: 10%~15%
  • price: 50~75
  • Formula: population * adoption_rate * price

Tips

  • Use descriptive variable names to make your calculations clearer
  • Start with wider ranges if you're more uncertain
  • You can use basic mathematical operators: + - * / ( ) ^
  • The calculator will show you different confidence intervals for your results
  • Use the share button to save and share your calculations
